# CALCULADORA EM JAVASCRIPT

## PREVIEW
![Preview](./assets/calculator.gif)

## TECNOLOGIAS USADAS:
- HTML5, CSS3 e JAVASCRIPT.

## PRINCIPAIS FUNCIONALIDADES:
- Operações básicas (+, -, *, /);
- Suporte a números decimais;
- Atualização dinâmica de resultado;
- Alternância de sinal (+/-);
- Tratamento de "Edge Cases" (divisão por zero, prevenir múltiplos operadores, etc.).

## CONHECIMENTO ADQUIRIDO:
- Controle e manipulação de estados sem frameworks;
- Conhecimento sobre Parsing de expressões matemáticas (resolução de expressões em duas etapas);
- Reforço sobre controle de eventos DOM;
- Tratamento de "Edge Cases";
- Separar adequadamente responsabilidades no código.

## FUTUROS UPGRADES:
- Suporte a teclado.
- Histórico de operações/equação.
- Refatoramento para REACT.
